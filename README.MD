# PixelBean 自动拼豆机
自制全自动拼豆机原型

## 当前版本
- 结构：CoreXY
- 供豆：单色
- 功能：单次拼 24 颗豆子

## 效率
- 单次拼 24 颗豆子，耗时 10 秒左右

## 硬件
- 主控：Arduino
- 驱动：步进电机 + 舵机
- 运动：CoreXY 结构
- 参考 https://www.bilibili.com/video/BV1cK411X7aa 大鱼diy写字机2022年版
- 参考 拼多多 点按式拼豆笔

## 软件
- G代码解析
- 坐标运动控制
- 单色落豆逻辑
