### **V1：原型验证**
- 目标：实现单色简单图案打印，验证机械运动、视觉标定、落豆的可行性。
- 交付：能打印出8*8的简单图案，验证核心功能。
- 原型功能
    - xy移动，精度0.1mm
    - 改造已有拼豆笔，支持快速使用
    - 上位机指挥单片机完成点阵打印
        - wasd 空格 控制拼豆机移动
        - 点阵->g代码程序，实现自动打印图案 

## 实际开发计划
1. 使用b站开源的大鱼写字机结构。coreXY结构， 
2. 机头使用磁吸舵机拼豆笔。
    - 手工上豆。多晃一晃
    - 磁吸定位机头和舵机连线， 实测对于拼豆无精度影响
3. 供料和机头一体
4. 视觉方案先用人眼定位。基本可用
5. 控制核心用ardiuno nano + python 上位机。
    - nano完成 串口处理和 gcode解析后， 作为完整版，不在更新固件
    - python上位机先用 cli + 键盘监听。 完成基本测试。
6. 持续优化速度/成功率
7. 随时更新，整理文档
8. 输出原型机演示视频
  
 
##### 【操作流程（3步）】
1. 开机 →  视觉标定
2. 导入18×18图片-> 上位机指挥下位机 → 检查完毕，电机启动按钮
3. 随时补充豆子

##### 【验收标准（必须全部满足）】
1. 运动：XY轴定位误差≤0.05mm，无丢步
2. 下料：单颗落豆成功率≥99.9% 
4. 视觉：标定后坐标误差≤0.1mm
5. 成品：18×18单色图完整，7次图片测试完成时间可接受， 成功率≥99%
