## sheild Vref 调试
所谓Vref电压调节，就是通过A4988上的电位器旋钮调节Vref参考电压。该电压将直接影响到步进电机在工作时流过线圈的电流强度。请注意，这一步操作非常重要，如果忽略这一步有可能会产生电机损坏的后果。
Vref = Imax X Rcs X 8
Rcs（看s1 s2电阻， 看一个就行） = 0.1 o
最大电流1.5a =》 vref = 1.5 x 0.1 x 8 = 1.2v

** 果然拿到手要调试， 起步0.5v 调到1.18v
** a4889模块右下角第二个口是vcc 可以供电
** a4889模块右下角第1个口是gnd 可以共地
