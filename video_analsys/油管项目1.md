More Details:
Total project time was about 3 months (Jan 2021 - April 2021)
Base used to be a cnc machine (BobsCNC Evolution 4)
One big change was moving from an all in one setup (v1), to 3 separate stations (bead refiller, pick/place machine, fuser). The reason I separated the bead refiller was that it was extremely loud in v1, caused vibrations throughout the machine, beads would also occasionally get stuck. So having this run separately and only when I needed it was a huge benefit. The fusing system was originally part of the pick/place machine but it was hard to keep the bed leveled (needed some level detection hw). Also, when fusing large pieces, the plastic smell was pretty bad and probably not too healthy. I moved the piece to a filtered tent in the garage for these reasons
